# Future Roadmap

## Near Term

- Add CODEOWNERS-based subsystem ownership inference.
- Add redaction for secrets and customer identifiers.
- Add richer stack-frame mapping for transpiled code.
- Add test generation for proposed patches.
- Add configurable scenario replay suites.

## Mid Term

- Integrate OpenTelemetry traces and spans.
- Link incidents to GitHub issues and pull requests.
- Add CI replay for proposed patches.
- Add service dependency graph ingestion.
- Add Slack incident channel summaries.

## Long Term

- Kubernetes pod watcher.
- Rollback recommendation engine.
- Multi-service blast radius propagation.
- Incident trend analytics.
- SLO-aware severity scoring.
- Human feedback loop for confidence calibration.

## Product Principle

Every roadmap item must reduce MTTR, reduce cognitive load, improve safety, or improve operational memory. Anything else is deferred.
